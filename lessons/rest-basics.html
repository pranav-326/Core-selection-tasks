REST & HTTP Basics

Learning objectives

  • Describe HTTP verbs: GET, POST, PUT/PATCH, DELETE
  • Interpret common HTTP status codes
  • Use Postman to test endpoints

Quick reference

GET
Read resources
POST
Create resources
PUT / PATCH
Update resources (PUT = replace, PATCH = partial)
DELETE
Remove resources

Common status codes

  • 200 OK
  • 201 Created
  • 204 No Content
  • 400 Bad Request
  • 401 Unauthorized
  • 403 Forbidden
  • 404 Not Found
  • 500 Server Error

Postman example

Send a GET to http://localhost:3001/posts and inspect headers, response, and status code.

Code example (fetch vs XHR)

// fetch example
fetch('/posts')
  .then(r => r.json())
  .then(data => console.log(data));

// XHR fallback
const xhr = new XMLHttpRequest();
xhr.open('GET', '/posts');
xhr.onload = () => console.log(JSON.parse(xhr.responseText));
xhr.send();
