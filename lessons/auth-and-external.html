Auth & External APIs

Learning objectives

  • Simulate token-based auth and attach `Authorization` header
  • Integrate external APIs (JSONPlaceholder)
  • Understand limitations of public mock APIs

Attach auth token

function attachAuthToken(headers = {}) {
  const token = localStorage.getItem('auth_token');
  if (token) headers['Authorization'] = 'Bearer ' + token;
  return headers;
}

Example usage:

fetch(`${API_BASE}/posts`, {
  method: 'GET',
  headers: attachAuthToken({ 'Content-Type': 'application/json' })
})

External API (JSONPlaceholder)

Switch `API_BASE` to https://jsonplaceholder.typicode.com for external testing. Caveats:

  • POST/PUT/PATCH are faked and won't persist long-term
  • Auth headers are ignored by JSONPlaceholder
  • Rate limits or differences in response shape possible
