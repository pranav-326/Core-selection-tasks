Mock API & json-server

Learning objectives

  • Understand `db.json` shape
  • Run `json-server` and inspect endpoints
  • Customize `json-server` options like `--port`

db.json explained

Collections in `db.json` become RESTful endpoints:

  • `/posts` - array of post objects
  • `/users` - users
  • `/comments` - comments

Examples:

GET /posts
POST /posts
GET /posts/:id
PUT /posts/:id
DELETE /posts/:id

Custom options

Run on a different port:

json-server --watch db.json --port 3001
