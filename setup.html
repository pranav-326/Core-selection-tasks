REST Workshop

Setup

Learning objectives

  • Install Node & npm
  • Run `json-server` mock API
  • Serve static site locally
  • Use Postman or browser devtools to inspect requests

Prerequisites

  • Node.js & npm (LTS recommended)
  • VS Code (optional)
  • Postman or HTTP client (optional)

Commands

npm install
npm run mock-api
npm run dev

Open http://localhost:5000 for the site and http://localhost:3001/posts for the API.

Switching to JSONPlaceholder (external API):

  1. Edit `js/config.js` and set: 
    export const API_BASE = 'https://jsonplaceholder.typicode.com';
export const USE_MOCK = false;
  2. Reload `demo.html`. Note: POST/PUT requests to JSONPlaceholder are simulated and not persisted.

Postman Quick Example

GET posts:

GET http://localhost:3001/posts

POST example body (JSON):

{
  "title": "Hello",
  "body": "Workshop post body",
  "userId": 1
}
